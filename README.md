# RAII Wireless

RAII-Wireless is a customized variant of [Sweep Bling LP](https://github.com/davidphilipbarr/Sweep), featuring splayed columns and various PCB and footprint updates.

## Modifications

* Applied splay to the ring and pinky columns
* Added support for Kailh Choc V2
* Compatible with Nice!Nano v2 or Pro Micro MCUs
